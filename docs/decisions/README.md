# Architecture Decision Records

This directory holds Blipbird's Architecture Decision Records (ADRs) — durable,
dated records of non-trivial decisions, especially the **provider
licensing/retention gate** answers referenced throughout `PLAN.md` §4.6 and
§16.

A disclaimer never replaces legal certainty. When a §4.6 gate closes (or
explicitly fails), record the answer here (without credentials or confidential
contract text) as an ADR and reference it from `PLAN.md` Appendix B.

## How to write one

1. Copy [`TEMPLATE.md`](TEMPLATE.md) to `NNNN-short-title.md`, incrementing the
   number (zero-padded).
2. Fill it in.
3. Add a row to the index below.
